# 📊 WeCom Task Manager - 表格自动初始化功能

**版本**: v1.4.0  
**更新时间**: 2026-03-27 17:35 CST

---

## 🎯 功能概述

**自动初始化表格** - 首次使用时自动创建完整的表格结构，无需手动配置。

### 核心特性

1. **首次使用自动检查** - 创建任务前自动检查表格完整性
2. **缺失字段自动补齐** - 自动添加缺失的必需字段
3. **CLI 命令支持** - 手动检查和初始化表格
4. **智能提示** - 区分必需字段和可选字段

---

## 📋 字段分类

### 必需字段（21 个）✅

这些字段通过 API 自动添加：

| 分类 | 字段 | 类型 |
|------|------|------|
| **基础字段** (9 个) | 任务 ID、任务名称、任务描述、任务类型、优先级、负责人、状态、截止时间、进度 | TEXT/SINGLE_SELECT/DATE_TIME/PROGRESS |
| **时间字段** (3 个) | 创建时间、实际开始时间、实际完成时间 | DATE_TIME |
| **工时字段** (1 个) | 预计工时 | NUMBER |
| **扩展字段** (8 个) | 前置依赖、阻塞原因、风险等级、输出物、备注、协作成员、关联目标、验收信息 | TEXT/URL/USER/SINGLE_SELECT |

### 可选字段（2 个）⚠️

这些是自动计算字段，需要手动配置：

| 字段 | 类型 | 说明 |
|------|------|------|
| **最后更新时间** | DATE_TIME | 每次编辑时自动更新 |
| **实际工时** | NUMBER | 自动计算：(完成时间 - 开始时间) / 3600000 |

## 🔧 手动配置可选字段

### 配置"最后更新时间"

1. 打开企业微信智能表格
2. 点击「添加列」→「创建公式」
3. 字段名：`最后更新时间`
4. 公式：`NOW()`（每次编辑时自动更新）
5. 格式：日期时间

### 配置"实际工时"

1. 打开企业微信智能表格
2. 点击「添加列」→「创建公式」
3. 字段名：`实际工时`
4. 公式：`(实际完成时间 - 实际开始时间) / 3600000`
5. 格式：数字（保留 2 位小数）

## 🛠️ API 参考

### check_table_status()

检查表格状态。

```python
from table_initializer import check_table_status

status = check_table_status()
print(status)
# {
#     "exists": True,
#     "field_count": 21,
#     "missing_fields": [],
#     "optional_missing": ["最后更新时间", "实际工时"],
#     "is_complete": True
# }
```

### initialize_table()

初始化表格（添加缺失字段）。

```python
from table_initializer import initialize_table

result = initialize_table()
print(result)
# {
#     "success": True,
#     "added_fields": ["关联目标", "验收信息"],
#     "failed_fields": [],
#     "message": "添加 2 个字段，失败 0 个"
# }
```

### ensure_table_initialized()

确保表格已初始化（在创建任务前调用）。

```python
from table_initializer import ensure_table_initialized

if ensure_table_initialized():
    print("表格已就绪")
else:
    print("表格初始化失败")
```

## 🐛 故障排查

### 问题 1：字段添加失败

**现象**：
```
❌ 添加失败：system error
```

**原因**：企业微信 API 临时故障

**解决**：
1. 等待 1-2 分钟后重试
2. 或手动在企业微信 APP 中添加

### 问题 2：找不到配置文件

**现象**：
```
⚠️ 配置文件不存在
```

**解决**：
```bash
# 确保从正确的目录运行
cd ~/.openclaw/skills/wecom-task-manager/scripts
python3 table_initializer.py check
```

### 问题 3：字段数量不对

**现象**：必需字段不足 21 个

**解决**：
```bash
# 运行初始化
python3 table_initializer.py init
```
